(初期は空、運用開始後 skill が自動追記)

## 運用ルール

### skill側の処理

1. 対話内容から候補タグ生成
2. 「正式登録」と完全一致 → そのまま使用
3. 正式登録 or Pending と類似(セマンティック判定) → 既存タグ採用
4. 完全不一致 → Pending に追記、使用回数1で記録
5. 既存Pendingタグの再利用 → 使用回数インクリメント

### Kaina側の月次整理

- **正式登録への昇格基準**: 使用回数3回以上 + 概念として独立してる
- **統合**: 表記ゆれ・類義語があれば代表タグに集約(例: `#配送` → `#配達`に統合)
- **削除**: 1ヶ月使われない / 一過性の固有名詞 / Pending放置

### 命名規則

- ハッシュ記号 `#` を頭に付ける(Obsidianタグ形式)
- 漢字・ひらがな・カタカナOK、半角英数も可
- スペース・記号は使わない(`#取引先対応` ○ / `#取引先 対応` ×)
- 取引先名は正式名称または略称を統一(辞書登録時にKainaが決める)
